# Project Overview

- This project is based on analyzing a customer transaction dataset using SQL and Excel.
- The objective of this project is:
- To analyze customer purchasing behavior
- To extract insights using SQL queries
- To present results using Excel analysis and dashboard

# Files Included in Project

- CSV File → Raw dataset containing customer transaction data
- SQL Files → Queries used for data analysis
- Excel Analysis File → Contains processed results
- Excel Dashboard Snapshot → Final visualization

# Work Performed
**Data Handling**
- Imported the CSV dataset into SQL
- Structured and prepared data for analysis
  
**SQL Analysis**
Performed analysis using queries such as:
- Aggregations (SUM, AVG)
- GROUP BY operations
- Filtering conditions
  
**Excel Analysis**
- Exported SQL query results into Excel
- Conducted further analysis using tables and calculations
  
**Dashboard Creation**
- Developed an Excel dashboard to present insights visually

# Key Insights 
- Total Revenue Generated: 4,915,544
- Top Performing Product Category: Electronics
- Top Revenue Generating City: Istanbul
- Average Delivery Time: ~6.5 days
- Average Customer Rating: ~3.9

# Observations
- The Electronics category contributes the highest share of revenue
- Sales are concentrated in specific high-performing cities
- Average delivery time ranges between 6–7 days
- Customer ratings are moderate, indicating potential areas for improvement

- Data extraction using SQL

# Project Walkthrough

I worked on a customer behavior analysis project using a real-world transaction dataset in CSV format. I began by importing the data into SQL, where I performed data analysis using queries involving aggregations and grouping. Through this process, I derived key insights such as total revenue, top-performing product categories, and city-wise performance.
For instance, the Electronics category generated the highest revenue, with total revenue reaching approximately 4.9 million.
After completing the SQL analysis, I exported the results into Excel for further processing and created a dashboard to visually present the insights.
